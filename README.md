# My Phasmophobia Ghost Ideas
A phasmophobia fan project made by me!

Became a small project for myself to create my ghost ideas but decided to make this my first public repository.

The Jiangshi was made before the Kormos was introduced! Both ghosts are nearly the same. Although, my idea for the ghost is slightly different with the Jiangshi being distracted by turned on spiritboxes. 
So it will stop chasing the player footsteps if a turned on spirit box is nearby (Currently only capable for 1 spirit box, functionality for multiple spirit boxes are being added)

2nd Ghost idea will introduce a ghost that is the opposite of the banshee where it will protect 1 player (Not my idea, this is from "Phasmophobia Devs Rate Your Ideas" video on yt)
